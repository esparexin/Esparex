# Security Policy

## Supported Versions

Currently, only the latest version of the repository is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Please do not open a public issue for a security vulnerability. 

Instead, report it via GitHub Security Advisories or contact the repository administrators directly.

We will acknowledge your report within 48 hours and work on a fix promptly.
