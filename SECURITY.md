# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security issues to `security@esparex.in`. We will respond within 48 hours.

## Disclosure Policy

1. Do not disclose vulnerabilities publicly until a fix is released.
2. We aim to resolve critical issues within 5 business days.
