# Esparex Platform — Component Conventions

## 1. File Naming
- Use PascalCase for component filenames.
- Exported default should match filename.

## 2. Import Hygiene
- No deep relative imports in components.
- Use path aliases for shared code.

## 3. SSR/CSR Rules
- SSR exceptions must be documented in code and README.
