# Esparex Platform — Dead Code & Comment Conventions

## 1. Dead File Removal
- Delete files with zero active imports or references.
- Confirm with directory listing and import search before deletion.

## 2. Comment Hygiene
- Remove obsolete TODO, FIXME, HACK comments.
- Retain migration and deprecated comments for historical clarity.

## 3. Import Hygiene
- Remove unused imports regularly.
- Use lint tools to enforce import cleanliness.
