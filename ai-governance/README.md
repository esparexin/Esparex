# AI Governance

This folder is the only versioned home for AI governance in this repository.

Use these files in order:

1. [SSOT.md](./SSOT.md) — the only authoritative AI governance source of truth
2. [SOP.md](./SOP.md) — the operational procedure for AI-assisted changes
3. [AI_CONTEXT.json](./AI_CONTEXT.json) — machine-readable inventory, tool mapping, and audit state
4. [PROMPT_TEMPLATE.md](./PROMPT_TEMPLATE.md) — the unified prompt template for IDE agents
5. [AUDIT_REPORT.md](./AUDIT_REPORT.md) — the repository audit of AI brains, duplicates, conflicts, and dead files
6. [MIGRATION_PLAN.md](./MIGRATION_PLAN.md) — the consolidation and cleanup plan

Rules and agent wrappers under `RULES/` and `AGENTS/` are compatibility aids only. They are not additional sources of truth.
