# AI Skills Library

This directory contains executable expertise for the AI.

## Skill Creation Rules
Do **not** create a new skill unless you comply with the `Skill Creation Rule` defined in `governance/GOVERNANCE.md`.

A Skill can only be created if:
* It has been used at least three times.
* It contains reusable expertise.
* It includes: Purpose, Scope, Checklist, Examples, and Anti-patterns.
* It is not already covered by another Skill.
